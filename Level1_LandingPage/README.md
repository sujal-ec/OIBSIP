## 📌 Objective

A landing page is one of the best web development projects for beginners.
This project demands a foundational understanding of HTML and CSS.
The goal is to build a visually impressive, well-structured landing page
with columns, sections, headers, footers, and proper alignments — ensuring
no elements overlap and CSS is used creatively.

## ✨ Features

- Sticky navigation bar with logo and enroll CTA button
- Hero section with tagline, stats grid, and dual action buttons
- Subjects section with 6 responsive subject cards
- "How It Works" — 4-step process section (dark themed)
- "Why Us" — feature highlights with founder quote card
- Student testimonials with star ratings
- Enrollment call-to-action banner
- Fully structured 4-column footer
- Fully responsive — mobile, tablet, and desktop

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Page structure and content |
| CSS3 | Styling, layout, animations |
| CSS Grid | Multi-column section layouts |
| Flexbox | Navigation and card alignment |
| CSS Variables | Consistent color theming |
| Google Fonts | Syne (headings) + Inter (body) |
